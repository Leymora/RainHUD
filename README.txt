RainHUD - Created by Netaro 2015


How to install
------------------------------------

1. Unzip RainHUD.zip

2. Place the RainHUD folder inside Program Files\Steam\steamapps\common\Team Fortress 2\tf\custom

Done!

(Optional) If you are using Windows then you can run the RainHUDHelper program. RainHUDHelper makes it easier
to change the settings of the HUD. Example: Change the color of the Health Value and Death Warning.

------------------------------------
Please note that on some computers; "Program Files" may look like "Program Files (x86)"
